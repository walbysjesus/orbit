# orbit
app de comunicacion satelital desarrollada por walbys rodriguez
Plataforma de Comunicación Satelital Inteligente

Orbit es un sistema de comunicación satelital diseñado para ofrecer conectividad continua, incluso en zonas sin cobertura celular. Combina tecnología de comunicación satelital, backend de alto rendimiento en Go, e interfaz móvil multiplataforma desarrollada en Flutter.
