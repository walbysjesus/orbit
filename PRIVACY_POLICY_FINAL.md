# PRIVACY POLICY - Orbit App

## Última actualización: 2024-06-20

Orbit ("nosotros", "nuestro", "la Compañía") opera la aplicación móvil Orbit (en lo sucesivo, "la aplicación").

Esta página le informa de nuestras políticas respecto a la recolección, uso y divulgación de datos personales cuando utiliza nuestra aplicación y de las opciones que tiene asociadas a esos datos.

### 1. INFORMACIÓN QUE RECOLECTAMOS

Recolectamos la siguiente información:

#### Información de Identificación Personal
- Nombre completo
- Dirección de correo electrónico
- Número telefónico
- Foto de perfil
- Información de autenticación (Firebase Auth)

#### Información de Comunicación
- Mensajes de chat (texto, imágenes, archivos)
- Registros de llamadas (audio y video)
- Duración de llamadas
- Información de estado de usuario (en línea/fuera de línea)

#### Información Técnica
- Dirección IP del dispositivo
- Información del dispositivo (modelo, sistema operativo, versión de la app)
- Registros de errores (Crashlytics)
- Tokens de notificación (FCM)

#### Información de Ubicación
- Ubicación general (recolectada solo si el usuario lo permite)

### 2. CÓMO USAMOS LA INFORMACIÓN

Utilizamos la información recolectada para:

1. Proporcionar y mejorar la aplicación
2. Facilitar comunicación entre usuarios (chat, audio, video)
3. Autenticar y verificar su identidad
4. Enviar notificaciones push
5. Analizar el rendimiento y usar (Firebase Analytics)
6. Monitorear errores y fallos (Crashlytics)
7. Cumplir con obligaciones legales

### 3. ALMACENAMIENTO Y SEGURIDAD DE DATOS

Sus datos se almacenan en:
- **Firebase Firestore**: Base de datos en la nube
- **Firebase Storage**: Almacenamiento de archivos y fotos
- **Firebase Authentication**: Autenticación y gestión de cuentas

#### Medidas de Seguridad:
- Encriptación en tránsito (HTTPS/TLS)
- Firestore Security Rules para control de acceso
- Autenticación de Firebase
- No compartimos datos con terceros sin su consentimiento

### 4. RETENCIÓN DE DATOS

- **Mensajes de chat**: Se mantienen indefinidamente
- **Registros de llamadas**: Se mantienen por 90 días
- **Datos de usuario inactivo**: Se pueden eliminar después de 12 meses sin actividad
- **Logs de errores**: Se retienen por 30 días

### 5. DERECHOS DEL USUARIO

Tiene derecho a:
- Acceder a sus datos personales
- Corregir información inexacta
- Solicitar la eliminación de sus datos (derecho al olvido)
- Exportar sus datos
- Objetar el procesamiento de sus datos

Para ejercer estos derechos, contacte a: support@orbitapp.com

### 6. TERCEROS

La aplicación utiliza los siguientes servicios de terceros:

#### Google Firebase
- Firebase Authentication
- Firebase Firestore
- Firebase Storage
- Firebase Messaging
- Firebase Analytics
- Firebase Crashlytics
- [Política de privacidad de Google](https://policies.google.com/privacy)

#### Servicios WebRTC
- Servidores STUN/TURN públicos para establecer llamadas P2P

### 7. CAMBIOS A ESTA POLÍTICA

Podemos actualizar esta Política de Privacidad de vez en cuando. La versión más reciente será siempre accesible desde la aplicación.

### 8. CONTACTO

Si tiene preguntas sobre esta Política de Privacidad, contáctenos en:
- Email: privacy@orbitapp.com
- Website: https://orbitapp.com/privacy
